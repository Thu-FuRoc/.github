## 🏠 关于我们

**清华大学未来机器人兴趣团队** 成立于 2004 年，是由教务处专项支持、校团委指导的高水平学生科创团队，汇聚来自不同院系的本科生与研究生。

团队致力于参加国际高水平赛事，推动具身智能领域机器人技术的研究与应用，构建开放、创新、充满活力的技术交流与实践平台。

课程立项**国家级一流本科课程虚拟仿真实验教学课**，获以下项目支持：

- 清华大学本科教学改革项目
- 实验室创新基金
- 大学生"追光计划"第四期

## 👥 三支代表队伍

### 🔥 火神队

> RoboCup Humanoid AdultSize | 成立于 2004 年

专注于类人机器人足球，核心研究方向为**平衡控制与动态步态算法**。

- 🏆 **RoboCup 2025 AdultSize 冠军** — 中国团队首次在该组别夺冠，刷新队史 21 年最佳纪录
- 🏆 RoboCup 2025 德国公开赛 AdultSize 冠军（9:0 零封对手）

### ⚡ TH-MOS队

> RoboCup Humanoid KidSize

专注于小型类人机器人的**视觉感知、快速决策与动作规划**。

- 🥈 **RoboCup 2025 KidSize 亚军** — 创队史 21 年最高排名，击败六届冠军法国 Rhoban 队
- 曾蝉联两届国内冠军，跻身全球四强

### 🏠 Tinker队

> RoboCup @home 家庭服务机器人

专注于**家庭服务机器人**，核心方向包括场景识别、语音交互与人机协作。

- 🏅 **RoboCup 2025 @home 全球八强** — 展现了在家用机器人领域的深厚积累

## 🚀 开源社区部

开源社区部于 **2026 年 4 月** 成立，是团队面向开源生态的核心力量。

### 📌 核心工作

| 方向 | 描述 |
|------|------|
| 📚 **资料沉淀** | 将前沿论文、技术博客、开源项目等内容沉淀为结构化中文资料，而非原始链接堆叠 |
| 🔬 **课题深入** | 在资料积累基础上选择具体课题深入，推进复现与实验 |
| 🤝 **开源贡献** | 推进项目协作与开源贡献，将阶段成果回流到资料库 |
| 🔄 **持续迭代** | 形成资料 → 课题 → 实验 → 回流的持续迭代闭环 |
